# 天地人·心性论

> Heaven, Earth, and Humanity · A Treatise on Heart-Mind and Nature

---

## 概述

《天地人·心性论》是一篇哲学短文，以中国古典心性论传统为根基，融合儒家、道家与宋明理学的思想气质，探讨**天、地、人**三者之间的关系，以及**人心**如何在天地之间安身立命。

全文以"天道无言而流行，地道承载而不执，人道立于其间而显生生之德"为主线，最终指向一个实践命题：**不向外求位，不向空逃苦，只在当下承当，于一念恻隐处见天心。**

## 中文原文

> 天道无言，却在万物中流行；
> 地道承载，却不执一隅；人处其中，方显生生之德。
> 安于天之无位，故不患"无"；身在地之有限，故能患"有"。
> 立于当下之边界，不逐外在之位，却在每一次恻隐中，让天理显性。
> 命本天授，却需人自去承担。
> 承担到最后，方知：
> 一切"患"皆从"空"中生，
> 一切"空"又皆依"患"而显。
> 天地不言，以生为道；
> 人心不昧，以觉为灯。
> 不向外求位，不向空逃苦，
> 只在当下承当，于一念恻隐处见天心。
> 此为人之所以立于世，合于天地之正道。

## 英文译本说明

本项目提供**两种英文译本**，并非简单的逐字翻译，而是同一思想的两种诠释路径：

### A. `ENG.form.becoming.txt` — 生成 / 流变风格

- **风格**：诗意、流动、体验式
- **特点**：保留中文原文的韵律与留白，以意象驱动，适合冥想式阅读
- **关键处理**：Tian → Heaven，Di → Earth，保持术语的神秘感与开放性

### B. `ENG.structure.flow.txt` — 结构 / 解析风格

- **风格**：分析性、定义式、哲学阐释
- **特点**：对核心概念（天、地、道、心）进行明确定义与展开，适合学术或研究性阅读
- **关键处理**：明确区分 Tian（nameless unfolding）与 Di（realm of form），引入 Dao、heart-mind 等术语解释

两种译本均分为 **8 段**，结构对称，可对照阅读。

## 核心概念

| 概念 | 含义 |
|------|------|
| **天 (Tian)** | 非人格化的主宰，而是无名之流行、万物生生不息的本源 |
| **地 (Di)** | 形器之域——名相、分别、概念、语言、思虑、身份、占有，一切有形者 |
| **人 (Humanity)** | 立于无界与有形之间，以觉性承当，以恻隐显天心 |
| **心 (Heart-Mind)** | 不昧之觉，内在之灯，非逃离形器，而是全然承载 |
| **患 / 空** | 忧患与虚空互为表里——患从空中生，空依患而显 |

## 思想渊源

本文的思想气质可追溯至：

- **《中庸》**："天命之谓性，率性之谓道"
- **张载《西铭》**："民吾同胞，物吾与也"
- **王阳明**："致良知"、知行合一
- **道家**：无为、自然、虚空与形器的辩证
